# MAGIK Offspecular Reflectometer

The MAGIK reflectometer complements the Polarized Beam Reflectometer and the Horizontal-Geometry Reflectometer in providing specialized capabilities for measurements of in-plane scattering in addition to world-class depth-profiling (with specular reflectivity) of biological, battery/electrochemical, polymer, and magnetic thin films.

## Description

MAGIK is located along the NG-D guide in the expanded guide hall (part of the NCNR's Expansion Initiative). It is an evolution of the AND/R reflectometer, which was previously located on the NG-1 guide.

### Key Capabilities
* **In-plane Scattering:** Specialized capabilities for off-specular measurements.
* **Depth-Profiling:** World-class specular reflectivity.
* **Polarization:** Polarized-beam measurements are possible with the use of a $^3\text{He}$ analyzer.

## Technical Resources and Tools

The instrument provides several interactive online tools for experiment planning and analysis:
* **Scattering Demos:** Interactive online scattering demonstrations.
* **Reflectometry Calculator:** Tool for calculating reflectometry parameters.
* **Alignment Simulator:** Simulator for instrument alignment.
* **Offspecular Scattering Planner:** Tool for planning off-specular experiments.

## Sample Environment and Support

The following sample environment controls are available:
* Refrigerators
* Furnaces
* Magnets
* Liquid cells
* Lasers

Additionally, on-site x-ray reflectometry is available for further sample characterization.
