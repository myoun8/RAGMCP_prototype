# MAGIK Offspecular Reflectometer

> DEPRECATION NOTICE: This page is no longer being updated and the information may be out of date.

Neutron spin filters can be applied to measurements requiring different field orientations. The use of Neutron Spin Filters (NSFs) on MAGIK allows the NSF to act as the analyzer for polarized beam measurements with a horizontal field.

For more details concerning neutron spin filters, refer to:
* General applications: {Chen03}
* Specific applications: {Chen04}

## Specifications for NSFs on MAGIK

| Parameter | Specification |
| :--- | :--- |
| Configuration | $^3\text{He}$ cell analyzer cell only |
| Capability | $^3\text{He}$ polarization flipping capability |
| Wavelengths | $\lambda = 5\text{ \AA}$ |
| Measurable Q range | Covers most of 2D PSD |
| Sample field | $H \le 0.7\text{ T}$ |
| Polarization correction | Four cross-section polarization correction |
| $^3\text{He}$ transmission | $\le 54\%$ (for desired state) |
| Flipping ratio | $\le 90$ |
