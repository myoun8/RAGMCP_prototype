# Data Reduction, Visualization and Analysis

## Getting your data
Raw data from non-proprietary experiments is available in the NCNR data repository at [https://doi.org/10.18434/T4201B](https://doi.org/10.18434/T4201B), as specified in the Data Management Plan.

## Available software
A number of software packages are available to NCNR users. Some must be used locally at the NCNR and others may be downloaded and used at home institutions. Most programs can be run locally on JAZZ. Programs that can be downloaded are indicated with a **(D)**.

### Crystallography
The following software were originally developed by Brian Toby while a staff member at the NCNR and are currently maintained at ANL:

*   **EXPGUI/GSAS**: Reduction and analysis of data for x-ray and neutron diffraction measurements (BT-1).
*   **CMPR**: Display, index and fit diffraction data.
*   **Other crystallography resources**.

### Reflectivity
Reduction, visualization and analysis of data from the PBR, MAGIK, and NG-7 reflectometers (with future support for CANDOR).

*   **Reductus**: Online data reduction.
*   **Refl1D (D)**: Advanced fitting with Python.
*   **Unpolarized neutron reflectivity**: Web calculator.
*   **Polarized neutron reflectivity**: Web calculator.
*   **Support**: [contact details omitted]

### Inelastic Neutron Scattering
*   **DAVE (D)**: For reduction, visualization and analysis of data from DCS, FCS, HFBS, SPINS, FANS (BT-4), BT2, BT7 and BT9.
    *   **Support**: [contact details omitted]
*   **XTREAT, TANQENS**: For reduction and analysis of data from FCS.
    *   **Support**: [contact details omitted]
*   **FIT (D)**: For fitting generic columnar-formatted datasets. (Available only inside firewall).
    *   **Support**: [contact details omitted]

### SANS
*   **IGOR* programs (D)**: For reduction and visualization of data from USANS, NG-B 30m SANS, NG-B 10m and NG-7 SANS.
*   **SasView (D)**: For analysis of data from USANS, NG-B 30m SANS, NG-B 10m and NG-7 SANS.
*   **Support**: [contact details omitted]

## Online calculators
Web-accessible tools for experimental planning and simulation:

*   **Neutron attenuation and activation** (BT-1)
*   **X-ray absorption calculator** (anl.gov)
*   **VSANS Online Calculator** (Available only inside firewall)

## DVA Team
The Data Visualization and Analysis (DVA) team is responsible for meeting the visualization and analysis software needs of the NCNR scientific staff and user community. Major current development efforts include the DAVE project and reflectivity software.
